Getting started

Welcome

Hello there! Thanks for buying Zien. This is a template with solid design especially crafted to help you built your Landing page for your Work Showcasing.

The concept of this template is Personal portfolio. This template uses Twiter's Bootstrap Framework and a custom font icons set It is highly recomended that you are familiar in a way with at least HTML5 / CSS3 / Basic JQuery.

So let's jump directly to it, and start everything from scratch.

Downloading Zien

Once you download the zip file, unzip it. You should have the following folders & files by now.

Folder Contents
Documentation Documentation and help instructions.
ZienTemplate Folder Zien Template Folder

Installation

Install the template via ftp :

  1. Connect to your website host, using some of many FTP clients, for example FileZilla.
  2. Go to your default server folder, and copy Zien into that folder.

Setting your fonts

To use google fonts, head over to Google's Fonts and add whichever font you want to your collection. Once you finished, follow the instructions and place the generated link in your HTML's file <head> tag.
The link should look like this : 

<link href='http://fonts.googleapis.com/css?family=Lato' rel='stylesheet' type='text/css'>
Next, go to style.css (css/style.css), and replace the font family, with the font family you just copied. 
 body {
font-family: 'Your Font', verdana;
font-weight: 400;
}

General layout

Every page is built with the latest standards of HTML5, validated with the W3C HTML Validation tool and it is compatible with the latest Twitter's Bootstrap.

All pages have 3 main parts:

  1. Navigation menu 1 ( For Landing page )
  2. Navigation menu 2 ( For Blog Page and Single Post Page )
  3. Sections with content (Including splash images/sections)
  4. Footer
